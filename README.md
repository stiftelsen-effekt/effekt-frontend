# effekt-frontend
Frontend for stiftelsen effekt

# Setup
To setup dev enviroment

Install NPM packages

```
npm install
```

Compile sass files to css files

```
gulp
``` 

Watch sass files for changes, and compile them to css when changed

```
gulp watch
```
